filmOpedia Documentation

not just another internet movie database

Objective:

Building the server-side part of a internet movie database, where users can register, sign in and save information about their favorite movies. A bit like the yellow one with the big black letters, but different.

User Stories:

As a user, I want to receive information on movies, directors and genres so that I can learn more about movies I watched or find out what I want to watch next.

As a user, I want to create a profile so that I can save my favorite movies.

Essential Features:

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister
  • More functions could possibly follow...

This project will be built with the following tools

  • The API is a Node.js and Express application
  • The API uses the REST(ful) architecture
  • The API contains of more than 3 middleware modules
  • The API uses a 'package.json'-File
  • The database is built using MongoDB
  • The business logic was modeled with Mongoose
  • The API provides movie information in JSON Format
  • The API includes user authentication and authorization code
  • The API includes data validation logic
  • The API meets data security regulations
  • The API source code is deployed to a publicly accessible platform
  • The API is deployed with Heroku

Description of endpoints

Request URL Method Body data Response
Display Welcome Page to the user / GET None A text message welcoming the user
Get a list of all movies /movies GET None JSON object containing data about all movies
Get a movie by title /movies/:title GET None A JSON object containing data(description, genres, director, image URL) about the selected movie
Get info about a genre /movies/:genre GET None A JSON object contain data about a genre
Get a list of directors /directors GET None JSON object containing all the directors
Get a director by name /directors/:name GET None A JSON object containing data about the director (name, bio, birthyear and death year)
Register a new user /users POST A JSON object holding data about the new user to register A JSON object containing data about user that has been registered
Deregister an existing user /users/:username DELETE None A text message indicating user was successfully removed
Update the username of an existing user /users/:username PUT None A text message indicating that the username was updated successfully
Add movie to the list of favorites /users/:username/favoritemovies POST A JSON Object holding data about the movie to add to the list of favorites A text message that the movie was added to the list of favorites
Remove movie from the favorite list /users/:username/favoritemovies/:movie DELETE None A text message indicating that a movie was successfully removed
